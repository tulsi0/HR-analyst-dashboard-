# TechiHub HR Analytics Dashboard (Attrition & Workforce Insights)

##  Project Overview  
This project presents a comprehensive **HR Analytics Dashboard built in Power BI** to analyze employee data across workforce distribution, attrition trends, and compensation-performance relationships.  

The dashboard helps HR teams identify **key drivers of attrition, workforce patterns, and performance gaps** for better decision-making.

---

##  Objectives  
- Monitor workforce metrics  
- Analyze attrition and retention trends  
- Identify reasons for employee exits  
- Evaluate compensation vs performance  
- Understand impact of satisfaction, tenure, and work mode  

---

##  Dataset Description  
The dataset consists of multiple tables:
- Employees  
- Performance Reviews  
- Salary History  
- Training  

### Key Features:
- Demographics (Age, Gender)  
- Department & Job Levels  
- Salary & Salary Increase %  
- Performance Scores  
- Work Mode (Remote, Hybrid, On-site)  
- Attrition Status & Exit Reasons  

---

##  Data Cleaning & Transformation   
- Standardized categorical data  
- Created calculated columns (Tenure, Salary Bands)  
- Built relationships across tables  
- Used DAX for KPIs and calculations
